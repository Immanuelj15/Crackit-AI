# 🚀 CrackIt AI - The Ultimate Interview & Coding Preparation Platform

CrackIt AI is a premium, AI-powered platform designed to help developers ace technical interviews. From realistic AI mock interviews to gamified coding challenges, CrackIt AI provides a comprehensive environment to sharpen your skills and land your dream job at top tech companies.

![CrackIt AI Banner](https://via.placeholder.com/1200x400/0f172a/38bdf8?text=Ace+Your+Next+Big+Interview+with+CrackIt+AI)

## ✨ Features

### 🎙️ AI Mock Interviews (Resume-Powered)
- **Realistic Simulations**: Experience interviews powered by advanced AI models tailored to your specific role and experience level.
- **Resume Parsing**: Upload your CV/Resume (PDF) and receive personalized questions based on your actual projects and skills.
- **Instant Feedback**: Get detailed scoring and actionable feedback on your answers to improve your performance.

### 💻 Advanced Coding Workspace
- **LeetCode-Style Editor**: A custom-built multi-pane coding editor with syntax highlighting and real-time execution.
- **Company Archives**: Access a rich collection of coding problems from top companies like Google, Amazon, and Meta.
- **Performance Charts**: Track your progress with visual "Beats" charts for runtime and memory usage.

### 🎮 Gamification & Engagement
- **CrackIt Coins**: Earn virtual currency for successful problem submissions and daily streaks.
- **Unlockable Content**: Use your coins to unlock expert editorials and premium company rounds.
- **Leaderboards**: Compete with other users and track your rank in the community.

### 📊 Performance Analytics
- **Personal Dashboard**: View your activity heatmap, streak history, and specialized skill metrics.
- **Goal Tracking**: Set and monitor your preparation goals.

## 🛠️ Tech Stack

### Frontend
- **Framework**: React.js
- **Styling**: Tailwind CSS (Glassmorphism & Dark Mode)
- **Animations**: Framer Motion
- **Charts**: Chart.js / React-Chartjs-2
- **State Management**: React Context / Hooks

### Backend
- **Server**: Node.js & Express.js
- **Database**: MongoDB (Mongoose ODM)
- **Authentication**: JWT & Firebase
- **File Handling**: Multer (Resume Uploads)

### AI Integration
- **Hugging Face Inference API**: For natural language processing and interview evaluation.
- **PDF-Parse**: For robust text extraction from resumes.

## 🚀 Getting Started

### Prerequisites
- Node.js (v20+ recommended)
- MongoDB account (Atlas or Local)
- API Keys: Hugging Face, Gemini, or Firebase (as per configuration)

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/Immanuelj15/Crackit-AI.git
   cd Crackit-AI
   ```

2. **Setup the Server**
   ```bash
   cd server
   npm install
   # Create a .env file with your credentials (PORT, MONGO_URI, JWT_SECRET, etc.)
   npm run dev
   ```

3. **Setup the Client**
   ```bash
   cd ../client
   npm install
   npm run dev
   ```
